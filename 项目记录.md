## 📊 安全架构

### 加密流程
```
用户输入 → 验证 → 派生密钥 → 加密 → 生成密文
```

**密钥派生**（PBKDF2）：
```
1.  派生随机 salt (16 bytes)
2. 派生随机 IV (12 bytes)
3. PBKDF2 迭代（100,000次迭代）
```

**消息格式**：
```
timestamp:message
```
